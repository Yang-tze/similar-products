## API Documentation
- GET: '/products'
- POST: '/products'
- PUT: '/productst'
- DELETE: '/products'


### Installing Dependencies

From within the root directory:
- Navigate to root directory
- Run npm install 

To run dev enviroment/webpack:
- Run npm build 

To run server:
- Run npm start
